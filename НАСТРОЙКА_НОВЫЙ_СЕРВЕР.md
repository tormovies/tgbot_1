# Настройка бота на новом сервере (website.com.ru)

**Путь к сайту:** `/home/admin/domains/website.com.ru/public_html`  
**Папка бота:** `/home/admin/domains/website.com.ru/public_html/tgbot98`

Админка: **https://website.com.ru/tgbot98/admin/** (или http — как настроен сайт)

---

## 1. Создать папку и права

```bash
mkdir -p /home/admin/domains/website.com.ru/public_html/tgbot98/data
chmod 755 /home/admin/domains/website.com.ru/public_html/tgbot98
chmod 755 /home/admin/domains/website.com.ru/public_html/tgbot98/data
```

Если нужен владелец — веб-сервер (подставь пользователя, если другой):

```bash
chown -R www-data:www-data /home/admin/domains/website.com.ru/public_html/tgbot98
```

Или, если от имени admin:

```bash
chown -R admin:admin /home/admin/domains/website.com.ru/public_html/tgbot98
```

---

## 2. Положить файлы бота

```bash
cd /home/admin/domains/website.com.ru/public_html
git clone https://github.com/tormovies/tgbot_1.git tgbot98
cd tgbot98
```

---

## 3. Конфиг

```bash
cp config.sample.php config.php
nano config.php
```

Заполнить: **BOT_TOKEN**, **DEEPSEEK_API_KEY**, **ADMIN_LOGIN**, **ADMIN_PASSWORD**. Сохранить.

---

## 4. Запуск бота

```bash
cd /home/admin/domains/website.com.ru/public_html/tgbot98
nohup php bot.php >> data/bot.log 2>&1 &
```

Проверка: `pgrep -af bot.php`

---

## 5. Проверка в Telegram и админке

- В Telegram написать боту **/son** → должен ответить.
- В браузере: **https://website.com.ru/tgbot98/admin/** — войти по логину/паролю из config.php.
